### ۳. فایل جدید: `store.tsx` — صفحه فروشگاه

تب‌بندی با ۲ تب:
- **تب «محصولات»**: لیست + جستجو + فیلتر + FAB افزودن (از products.tsx فعلی)
- **تب «ابزارها»**: شامل بخش‌های زیر:

#### ابزارهای موجود (از tools.tsx):
- API Key + کد نمونه
- ربات تلگرام
- ربات اینستاگرام
- صفحه وب رایگان

#### ابزارهای جدید (درخواست‌ها):
- **درخواست دستگاه کارتخوان**: فرم انتخاب بانک + نوع دستگاه + آدرس تحویل + توضیحات → ارسال درخواست
- **درخواست درگاه پرداخت**: فرم انتخاب بانک + نوع درگاه (وب/اپ) + آدرس سایت + توضیحات → ارسال درخواست
- **افتتاح حساب جدید**: فرم انتخاب بانک + نوع حساب + اطلاعات هویتی → ارسال درخواست

هر درخواست پس از ارسال: آنیمیشن موفقیت + شماره پیگیری + دکمه بازگشت

### ۴. فایل جدید: `financial.tsx` — صفحه مدیریت مالی

#### خلاصه مالی بالا (کارت گرادیان)
- مانده کل حساب‌ها
- فروش امروز
- تسویه در انتظار
- نرخ موفقیت

#### دو سایتب بالای صفحه:

**تب ۱: حساب‌ها**
- لیست کارت حساب‌های بانکی:
  - آیکون بانک + نام بانک (با رنگ اختصاصی)
  - شماره حساب مسکون
  - مانده حساب
  - وضعیت
- **کلیک → صفحه جزئیات حساب**

**تب ۲: پایانه‌ها**
- لیست کارت پایانه‌ها با تگ نوع:
  - **نوع**: `POS` یا `IPG` (تگ رنگی بالای کارت)
  - **بانک**: نام بانک با رنگ اختصاصی
  - **شناسه**: شماره سریال / شناسه ترمینال
  - **وضعیت**: فعال/غیرفعال
  - **آمار امروز**: تعداد تراکنش + مبلغ
- **کلیک → صفحه جزئیات پایانه**

---

### ۵. فایل جدید: `account-detail.tsx` — جزئیات حساب

- **هدر**: آیکون بانک + نام بانک + شماره حساب + مانده
- **نوار اقدامات**: مشاهده تراکنش‌ها / گزارش ماهانه
- **آمار**: ورودی امروز / خروجی امروز / مانده
- **لیست تراکنش‌ها**: فیلتر ورودی/خروجی/همه
  - هر ردیف: نوع (ورود/خروج) + شرح + مبلغ + تاریخ
  - مبالغ ورودی سبز، خروجی قرمز
- **نمودار گردش حساب**: Line Chart ماهانه

---

### ۶. فایل جدید: `terminal-detail.tsx` — جزئیات پایانه

- **هدر**: تگ نوع (POS/IPG) + نام بانک + شماره سریال
- **نوار اقدامات سریع**:
  - **فعال/غیرفعال**: Toggle با تأیید
  - **ثبت خرابی**: رفتن به صفحه تیکت
  - **تنظیمات**: Sheet تنظیمات
- **آمار**: تراکنش امروز/هفته/ماه + نرخ موفقیت
- **گزارش عملکرد**: ۳ تب زمانی:
  - **روزانه**: Bar Chart ۲۴ ساعت اخیر
  - **هفتگی**: Bar Chart ۷ روز
  - **ماهانه**: Bar Chart ۳۰ روز
- **لیست تراکنش‌ها**: فیلتر موفق/ناموفق/همه
- **Sheet تنظیمات**: نام مستعار + اطلاعات پشتیبانی

---

### ۷. فایل جدید: `terminal-ticket.tsx` — ثبت تیکت

- **نوع تجهیز**: پیش‌انتخاب شده (POS/IPG + نام بانک)
- **نوع مشکل**: Dropdown (خرابی سخت‌افزاری / خطای نرم‌افزاری / عدم اتصال / تراکنش ناموفق / مشکل تسویه / سایر)
- **اولویت**: کم / متوسط / زیاد / فوری
- **توضیحات**: Textarea
- **شماره تماس**: پیش‌پر شده
- **ارسال** → آنیمیشن موفقیت + شماره تیکت
