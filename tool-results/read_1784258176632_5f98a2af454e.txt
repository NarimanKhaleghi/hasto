     3→## Project Overview
     4→**هستو (Hasto)** — A comprehensive Persian fintech payment platform MVP.
     5→- B2C: Mother wallet / personal finance app with Direct Debit, payments, financial management, services, contracts
     6→- B2B: Business panel with payment links, products, tools (API, bots, web page)
     7→- Design: Tejarat Blue (#034ea2), Vazirmatn font, RTL, Glassmorphism, Dark+Light mode
     8→- Tech: Next.js 16, TypeScript, Tailwind 4, shadcn/ui, Zustand, Recharts
     9→
    10→## Architecture
    11→- Single-page app (only `/` route) with Zustand store managing screen navigation
    12→- Mock data only (no backend)
    13→- Phone-frame wrapper on desktop, full-screen on mobile
    14→- Mode switcher: B2C (user app) ↔ B2B (business panel)
